## 3. 场景优先级规则

每个场景按以下维度评分：

- 影响度（1-3）：失败后的业务影响。
- 发生概率（1-3）：基于复杂度、变更频率与依赖风险。
- 可发现性（1-3）：失败是否容易在早期被发现。

优先级建议：

- P0：总分 >= 7，或直接影响收入/安全/合规。
- P1：总分 5-6。
- P2：总分 <= 4。
