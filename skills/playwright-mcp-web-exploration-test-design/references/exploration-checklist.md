# 网页探索检查清单

## 页面级探索

- 确认页面目标与用户任务。
- 识别主 CTA、次 CTA、危险操作、返回/取消出口。
- 识别进入前置条件：登录态、权限、上游数据。
- 识别可见与隐式校验规则。

## 表单与输入行为

- 校验必填/选填字段。
- 校验边界条件：长度、数值范围、日期范围、格式掩码。
- 校验非法字符与注入类输入处理。
- 校验防重复提交与幂等行为。
- 校验错误提示位置与可理解性。

## 数据与状态行为

- 校验空态、加载态、成功态、失败态。
- 校验数据刷新与陈旧数据处理。
- 校验分页/筛选/排序的状态保持。
- 校验乐观更新与确认更新的一致性。
- 校验接口失败后的回滚或恢复机制。

## 导航与连续性

- 校验深链接与刷新恢复能力。
- 校验浏览器前进/后退行为。
- 校验未保存变更提示。
- 校验跨模块跳转后的上下文完整性。

## 权限与安全相关交互

- 校验未授权/无权限时的行为。
- 校验不同角色下控件与数据可见性。
- 校验会话过期与重新登录链路。
- 校验敏感操作的二次确认机制。

## 跨上下文检查

- 至少覆盖一个移动端视口和一个桌面端视口（如适用）。
- 校验日期/时间/货币在语言与时区维度的显示正确性。
- 校验基础可访问性：焦点顺序、键盘可达性、控件标签清晰度。

## 证据记录

- 每个结论记录：页面 URL、动作、预期结果、实际结果。
- 每个问题记录可复现步骤。
- 每个高风险结论保留最小充分证据。
